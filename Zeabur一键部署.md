# 🚀 Zeabur 超简单部署（像打卡系统一样简单）

## 只需3步，3分钟完成！

---

### 第1步：访问Zeabur

打开：**https://zeabur.com**

点击 **"Sign in"** → **"Continue with GitHub"**

---

### 第2步：导入项目

1. 点击 **"Create New Project"**
2. 点击 **"Add Service"** → 选择 **"Git"**
3. 选择仓库：`shihaowei49-cloud/wallet-management-system`
4. 点击 **"Import"**

**就这样！** 不需要设置Root Directory！

---

### 第3步：设置环境变量

点击服务 → **"Variables"** → 添加2个变量：

```
PORT=5001
NODE_ENV=production
```

点击 **"Deploy"**

---

## ✅ 完成！

等待3-5分钟，部署完成后：

1. 点击生成的域名，例如：`https://xxx.zeabur.app`
2. 登录：
   - 用户名：`admin`
   - 密码：`admin123`

---

## 就这么简单！

和您的打卡系统一样，**一个服务，一次部署**！

不需要分开部署前后端了 🎉
