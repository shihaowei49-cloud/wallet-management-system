# 🚀 不用控制台的部署方案

由于SSH自动连接存在问题，这里提供几个**不需要腾讯云Web控制台**的替代方案。

---

## ✅ 方案1：一键部署URL（最简单）

### 步骤：

**1. 通过任何方式连接到服务器**（SSH客户端、命令行等）

**2. 运行一条命令：**

```bash
curl -fsSL https://raw.githubusercontent.com/shihaowei49-cloud/wallet-management-system/main/自动拉取部署.sh | sudo bash
```

### 这个命令会自动：
- ✅ 检查并安装 Node.js、Git、PM2、Nginx
- ✅ 从GitHub克隆/更新代码
- ✅ 安装依赖
- ✅ 构建前端
- ✅ 启动后端
- ✅ 配置Nginx
- ✅ 设置开机自启

### 完成后访问：
`http://43.133.241.218`

---

## 🔧 方案2：使用第三方SSH客户端

### Mac/Linux：

使用系统自带的SSH命令（在您的本地终端运行）：

```bash
ssh administrator@43.133.241.218
# 输入密码：Helen123!@#
```

登录后运行：
```bash
curl -fsSL https://raw.githubusercontent.com/shihaowei49-cloud/wallet-management-system/main/自动拉取部署.sh | sudo bash
```

### Windows：

**选项A：使用 PuTTY**
1. 下载 PuTTY：https://www.chiark.greenend.org.uk/~sgtatham/putty/latest.html
2. Host Name: `43.133.241.218`
3. Port: `22`
4. Connection type: `SSH`
5. 点击 Open
6. 用户名：`administrator`
7. 密码：`Helen123!@#`
8. 运行一键部署命令

**选项B：使用 Windows Terminal/PowerShell**
```powershell
ssh administrator@43.133.241.218
# 输入密码
```

## 🌐 方案3：远程桌面（如果是Windows服务器）

服务器开放了3389端口，可能是Windows服务器：

### Mac连接Windows远程桌面：

1. 从App Store安装 "Microsoft Remote Desktop"
2. 添加PC:
   - PC name: `43.133.241.218`
   - User account: `administrator`
   - Password: `Helen123!@#`
3. 连接后在服务器上操作

### Windows连接Windows远程桌面：

1. Win + R，输入 `mstsc`
2. 计算机：`43.133.241.218`
3. 用户名：`administrator`
4. 连接并输入密码：`Helen123!@#`

**连接后：**
如果是Windows服务器，需要不同的部署方式（安装IIS或使用Windows版Node.js）

## 🆘 如果仍然无法连接SSH

### 可能原因：

1. **安全组/防火墙未正确配置**
   - 需要在腾讯云控制台开放22端口（对于SSH）
   - 或开放3389端口（对于远程桌面）

2. **SSH服务未启动**
   - Windows服务器默认不开启SSH
   - 可能需要安装OpenSSH Server

3. **IP白名单限制**
   - 某些云服务器只允许特定IP连接

### 解决方法：

**最简单的方式：**
- 登录腾讯云APP（手机）
- 找到您的服务器
- 使用"Web Shell"或"VNC"功能登录
- 运行一键部署命令
