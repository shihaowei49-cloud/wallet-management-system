# 🚀 快速使用指南

## 5 分钟上手 Wallet Management System

### 第一步：启动系统

```bash
cd wallet-management-system
./start.sh
```

等待依赖安装和服务器启动（首次运行需要几分钟）

### 第二步：登录系统

1. 打开浏览器访问 `http://localhost:3000`
2. 输入测试账号:
   - 用户名: `admin`
   - 密码: `admin123`
3. 点击登录

### 第三步：创建你的第一个钱包

1. 点击左侧菜单 "我的钱包"
2. 如果没有钱包，会提示创建
3. 点击 "创建钱包" 按钮
4. 选择 "创建新钱包"
5. **重要**: 记下助记词（12个单词）
6. 勾选确认备份
7. 完成！

### 第四步：获取测试币

你的钱包已创建，但余额为 0。获取测试币：

**Sepolia 测试网水龙头**:
1. 复制你的钱包地址
2. 访问: https://sepoliafaucet.com
3. 粘贴地址，申请测试币
4. 等待 1-2 分钟
5. 回到系统，点击刷新查看余额

### 第五步：发送你的第一笔交易

1. 在 "我的钱包" 页面
2. 滚动到 "发送" 部分
3. 输入接收地址（可以用另一个测试地址）
4. 输入金额（例如 0.001 ETH）
5. 点击 "发送交易"
6. 等待确认（约 15-30 秒）
7. 查看交易哈希和区块浏览器链接

## 🎯 核心功能速览

### 1. 钱包功能
- **我的钱包** - 查看余额、发送/接收、查看私钥
- **交易记录** - 所有交易历史
- **网络切换** - 右上角可切换网络

### 2. NFT 功能
- **NFT 监控** - 查看所有 NFT
- **转账 NFT** - 点击 NFT 卡片上的发送按钮

### 3. DApp 浏览器
- 访问 Uniswap、OpenSea 等热门 DApp
- 在新窗口中打开

### 4. 管理功能
- **数据看板** - 统计和图表
- **用户管理** - 管理多个用户
- **余额监控** - 监控钱包风险

## ⚡ 快捷操作

### 快速切换网络
```
顶部导航栏 → 网络切换器 → 选择网络
```

### 导入已有钱包

```
交易记录 → 点击交易 → 在区块浏览器中查看
```

## 🔐 安全提示

1. **永远不要分享**:
   - 私钥
   - 助记词
   - 密码

2. **务必备份**:
   - 将助记词写在纸上
   - 存放在安全的地方
   - 建议多处备份

3. **测试网优先**:
   - 先在 Sepolia 测试网练习
   - 熟悉后再使用主网
   - 主网操作需要真实资金

## 🆘 遇到问题？

### 交易失败
- 检查余额是否充足
- 确认网络是否正确
- 等待网络不拥堵时重试

### 余额不显示
- 点击刷新按钮
- 检查网络连接
- 确认钱包地址正确

### 无法创建钱包
- 清除浏览器缓存
- 检查控制台错误
- 确认后端服务运行正常

## 💡 Pro Tips

1. **使用测试网**: 所有操作先在测试网练习
2. **备份助记词**: 丢失后无法找回
3. **小额测试**: 首次交易使用小金额
4. **验证地址**: 发送前仔细检查接收地址
5. **Gas 费用**: 可以根据网络拥堵调整
